Sistem Operasi DOS

DOS (Disk Operating System) adalah sistem operasi berbasis teks yang populer di era 1980-an. Microsoft mengembangkan versi terkenalnya yaitu MS-DOS, yang menjadi dasar Windows generasi awal.

Sumber: Wikipedia - MS-DOS

Sistem Operasi Windows

Windows dikembangkan oleh Microsoft dengan antarmuka grafis. Dirilis tahun 1985 dan terus berkembang hingga Windows 11 yang sangat populer di seluruh dunia.

Sumber: Wikipedia - Windows

Sistem Operasi Mac

Mac OS atau kini disebut macOS adalah sistem operasi buatan Apple sejak 1984. Terkenal karena tampilannya yang elegan dan kestabilannya di perangkat Mac.

Sumber: Wikipedia - macOS

Sistem Operasi Linux

Linux adalah sistem operasi open-source berbasis kernel Linux. Diciptakan oleh Linus Torvalds tahun 1991 dan kini digunakan di berbagai server dan perangkat pintar.

Sumber: Wikipedia - Linux

Sistem Operasi Android

Android dikembangkan oleh Google sebagai sistem operasi berbasis Linux untuk smartphone. Diluncurkan tahun 2008 dan kini mendominasi pasar global.

Sumber: Wikipedia - Android
