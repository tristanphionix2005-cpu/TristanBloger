Sistem Operasi DOS

DOS (Disk Operating System) adalah sistem operasi berbasis teks yang digunakan pada komputer IBM-PC pada awal tahun 1980-an. Microsoft mengembangkan versi terkenalnya yaitu MS-DOS, yang menjadi dasar bagi sistem operasi Windows generasi awal.

Sumber: Wikipedia - MS-DOS

Sistem Operasi Windows

Windows dikembangkan oleh Microsoft sebagai antarmuka grafis di atas DOS. Dirilis pertama kali pada tahun 1985, Windows berevolusi menjadi sistem operasi modern seperti Windows 10 dan 11 yang digunakan di seluruh dunia.

Sumber: Wikipedia - Windows

Sistem Operasi Mac

Mac OS adalah sistem operasi buatan Apple untuk komputer Macintosh, diperkenalkan pada tahun 1984. Kini dikenal sebagai macOS, sistem ini terkenal karena desain elegan, performa stabil, dan integrasi kuat dengan perangkat Apple lainnya.

Sumber: Wikipedia - macOS

Sistem Operasi Linux

Linux adalah sistem operasi open-source berbasis kernel Linux yang dibuat oleh Linus Torvalds pada tahun 1991. Banyak digunakan untuk server, komputer pribadi, dan perangkat mobile.

Sumber: Wikipedia - Linux

Sistem Operasi Android

Android dikembangkan oleh Google sebagai sistem operasi berbasis Linux untuk perangkat mobile. Dirilis pertama kali pada tahun 2008, Android kini mendominasi pasar smartphone dunia.

Sumber: Wikipedia - Android
